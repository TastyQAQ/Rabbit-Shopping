## 安裝

```bash
# 獲取專案
git clone https://github.com/TastyQAQ/Rabbit-Shopping
```

```bash
# 安裝套件
pnpm install
```

```bash
# 運行專案
pnpm build
```

## 主要工具

* Vue v3.4.15
* Vue Router v4.2.5
* Pinia v2.1.7
* element-plus v2.5.3
* axios v1.6.7
